# Radim Šimánek

Building privacy-first iOS apps that run AI on-device. Based in Zurich.

## What I'm building

- **[Ahimo](https://ahimo.app)** — privacy-first iOS health app. Pairs Apple Watch + HealthKit data with the context only you can provide (mood, stress, drinks, symptoms) and uses on-device LLMs (Apple Foundation Models or MLX: SmolLM3, Qwen, Llama) to surface real personal patterns. No cloud processing of health data. [Public docs →](https://github.com/Rad1m/ahimo-public)

- **[Falcata Run](https://falcatarun.com)** — Apple Watch-first sprint training app. Track reaction time, drive force, cadence, stride length, and gait symmetry at 100 Hz. Built for sprinters, coaches, and Masters athletes. [Public docs →](https://github.com/Rad1m/falcata-sprint-run-public)

## Day job

**Agentic AI engineering at Roche.** Multi-agent systems and AI workflows inside a regulated pharma environment — where reliability, traceability, and governance are not optional. The hard problems on this side feed directly into how I think about building AI products on the indie side: caching for cost control, structured output for safety, evaluation harnesses you can actually iterate against.
